# Overview:

## Getting started:

### Local version
    $ git clone ${repo}
    $ npm install
    $ npm start

### Prod version
    $ npm run build
    $ cd ~/build

### Storybook
    $ npm run storybook dev -p ${port}  // run
    $ npm run storybook build           // build
